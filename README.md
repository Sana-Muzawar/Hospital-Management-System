# Hospital-Management-System
Hospital Management System is developed using SQL to efficiently manage hospital operations. The system stores and organizes information related to patients, doctors, appointments, treatments, and billing. It helps streamline hospital data management, reduce manual record keeping, and improve data retrieval.

Features
1. Manage patient records and personal details
2. Store doctor information and department details
3. Schedule and track patient appointments
4. Maintain treatment and medical records
5. Manage billing and payment information

SQL Concepts Used
1. Database and table creation
2. Primary keys and foreign keys
3. CRUD operations (Create, Read, Update, Delete)
4. SQL joins for retrieving related data
5. Aggregate queries for reports
